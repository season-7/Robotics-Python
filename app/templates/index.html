Pi Drive

Use the keyboard direction to control the car. The car uses a Real-Automatic gearbox. If you want to reverse and are currently moving forward, press the back button twice and then accelerate backwards with the up key. If at standstill, press the back button once and accelerate backwards using the up key.
To move forward in reverse state, press the back key once to stop, then accelerate forward ussing the up key.

For a better experince, downloadour android app.
